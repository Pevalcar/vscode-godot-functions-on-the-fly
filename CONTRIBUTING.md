# Contributing

Thanks for helping improve Godot Functions On The Fly.

## Workflow

1. Create a branch for your change.
2. Run `npm install` once if dependencies are missing.
3. Make the change in `src/`.
4. Run `npm test` before opening a pull request.

## Guidelines

- Keep changes focused and small.
- Prefer minimal, testable edits.
- Preserve the existing GDScript and VS Code conventions in the project.
- Update `README.md` when you change user-facing behavior.

## Reporting issues

Include the GDScript snippet you tested, the VS Code version, and the exact behavior you expected.
